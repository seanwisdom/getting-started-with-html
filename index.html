The internet is a collection of servers that communicate via the protocol HTTP.  The internet is accessed from a personal computer through a browser.

HTML is short for Hypertext Mark Up Language and is the language most commonly used to write code for web pages.

The key component of HTML is an element.  An element is opened and closed by a tag.

Elements are classified as either inline or block.

A tag can also have an attribute.  For example, the anchor tag uses the href attribute to set the target of the anchor.
